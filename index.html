net-next-sim: ns-3 Direct Code Execution with Linux kernel

Download as .zip Download as .tar.gz View on GitHub

Quick start

 git clone git://github.com/direct-code-execution/net-next-sim.git
 cd net-next-sim
 make defconfig ARCH=lib
 make library ARCH=lib
 make testbin -C arch/lib/test
 make test ARCH=lib

see what's going on.

NOTE Old code available at non-default branches (sim-ns3-2.6.36-branch, sim-ns3-3.14.0-branch, etc) use different ARCH name, which is sim. You need to replace the above instruction lib to sim (e.g., make library ARCH=sim should be used for 2.6.36 branch).

What is this ?

net-next-sim is a shared library of Linux (net-next) kernel to integrate with userspace applications, such as ns-3 network simulator. It consists of sim architecture (located arch/sim/) to avoid touching the main part of Linux kernel.
